# Java Spring Professional - DevSuperior

Este repositório reúne todos os projetos e exercícios desenvolvidos ao longo do curso *Java Spring Professional*, oferecida pela [DevSuperior](https://devsuperior.com.br/).  
Cada branch representa uma etapa prática do curso, contendo implementações, desafios e anotações de aprendizado.

A proposta é manter um histórico versionado da evolução técnica durante a formação.

## Estrutura do Repositório

1. Implementação de componentes, inversão de controle e injeção de dependência no Spring Boot.
2. Mapeamento objeto-relacional (ORM) com hibernate e estrutura de projetos Spring Boot.
3. Desenvolvimento de APIs REST com arquitetura em camadas, operações CRUD, integração com banco relacional, validações e tratamento de exceções.
4. Consultas com SQL nativo, JPQL e recursos do Spring Data para acesso eficiente a dados.
5. Autenticação e controle de acesso com OAuth2 e JWT.
<!-- 6. Implantação e monitoramento em produção -->

> Cada branch possui seu próprio README explicando o contexto, e desafios resolvidos.

## Tecnologias e Ferramentas Utilizadas

- Java 17+
- Spring Boot
- Spring Data JPA / Hibernate
- Spring Web / REST
- Spring Security
- Maven
- Postman
- Git / GitHub
- Spring Tool Suite
