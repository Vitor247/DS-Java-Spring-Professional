# Componentes e Injeção de Dependência

Neste módulo são praticados os conceitos fundamentais de componentes, inversão de controle (IoC) e injeção de dependência. Como estruturar sistemas modulares, trocar dependências sem modificar o componente pai e utilizar frameworks para facilitar essa arquitetura. Inclui a criação de um projeto Spring Boot e a implementação prática de componentes. Finaliza com um desafio prático para consolidar o conteúdo.
