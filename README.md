# API REST, Camadas, CRUD, Exceções e Validações

Este capítulo aborda os fundamentos de APIs REST, seus padrões, estruturas de URL e práticas de resposta HTTP. Construção de uma aplicação em camadas usando DTOs, paginação e operações CRUD completas. Inclui tratamento de exceções, validações com Bean Validation e personalização das respostas de erro. Finaliza com a implementação de um CRUD de clientes no desafio.
